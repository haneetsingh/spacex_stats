# SpaceX Stats
```
Apollo, GraphQL and React app which uses SpaceX API to display its launch stats
```

## Quick Start
```
# Install dependencies (server & client)
npm install
cd client && npm install

# Server only
npm run server

# Client only
npm run client

# Both server & client
npm run dev

# Graphiql - http://localhost:5000/graphql
# React App - http://localhost:3000
```
